# 메신저 미리보기 이미지 가이드

## 권장 이미지 크기

- **카카오톡, 페이스북**: 1200 x 630 픽셀
- **트위터**: 1200 x 675 픽셀
- **최소 크기**: 600 x 315 픽셀

## 현재 상태

현재 `vl-logo.png`는 228 x 160 크기입니다. 더 나은 미리보기를 위해 큰 이미지를 추가하는 것을 권장합니다.

## 미리보기 이미지 추가 방법

1. **이미지 준비**
   - 1200 x 630 크기의 이미지 생성
   - 파일명: `og-image.png` 또는 `og-image.jpg`

2. **파일 저장**
   - `public/` 폴더에 저장

3. **index.html 수정**
   - `og:image` 태그의 경로를 새 이미지로 변경
   - 예: `content="https://vl-rn-sheet.vercel.app/og-image.png"`

## 카카오톡 미리보기 확인

1. [카카오톡 개발자 도구](https://developers.kakao.com/tool/clear/og)에서 링크 확인
2. 이미지가 제대로 표시되는지 검증

## 참고

- 이미지 파일 크기는 1MB 이하 권장
- JPG 또는 PNG 형식 사용
- 텍스트가 포함된 경우 가독성 확인

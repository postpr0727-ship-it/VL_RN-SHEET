# 건강센터 간호사 근무표 자동 생성 시스템

건강센터 간호사들의 근무표를 자동으로 생성하는 반응형 웹 애플리케이션입니다.

## 주요 기능

- 📅 **자동 근무표 생성**: 버튼 클릭 한 번으로 공평한 근무표 생성
- 🏖️ **연차 관리**: 개인 연차 입력 시 자동으로 OFF 배정 및 근무표 재조정
- 📱 **반응형 디자인**: PC와 모바일에서 모두 사용 가능
- ⚖️ **공평한 OFF 분배**: 평일/주말 OFF를 최대한 공평하게 분배

## 근무 시간대

- **DAY**: 07:00 - 16:00
- **MID-DAY**: 09:00 - 18:00
- **EVENING**: 12:00 - 21:00
- **NIGHT**: 21:00 - 07:00
- **OFF**: 휴무

## 근무 인원 편성

### 평일 (월~금)
- DAY: 2명
- MID-DAY: 2명
- EVENING: 1명
- NIGHT: 1명

### 주말/공휴일 (토/일/공휴일)
- DAY: 1명
- EVENING: 1명
- NIGHT: 1명

## 간호사별 근무 패턴

### A, B, C, D 간호사
- 평일: DAY, MID-DAY, OFF
- 주말/공휴일: DAY, EVENING, OFF

### E, F 간호사
- 평일: DAY와 EVENING을 주 단위로 번갈아 근무
- 주말/공휴일: 무조건 OFF

### G, H 간호사
- 평일/주말: NIGHT, OFF

## OFF 분배 규칙

- 1인당 주당 2회 OFF (4주 기준 8회)
- E, F 간호사는 주말/공휴일 OFF 포함으로 더 많을 수 있음
- 평일/주말 OFF를 최대한 공평하게 분배

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 사용 방법

1. **근무표 생성**: 페이지 로드 시 자동으로 현재 달의 근무표가 생성됩니다.
2. **월 변경**: "이전 달" / "다음 달" 버튼으로 원하는 월을 선택합니다.
3. **연차 입력**: 
   - 간호사 선택
   - 날짜 선택
   - "추가" 버튼 클릭
   - "근무표 생성" 버튼을 클릭하여 연차를 반영한 새로운 근무표 생성
4. **근무표 확인**: 
   - 달력 뷰에서 날짜별 근무 현황 확인
   - 테이블 뷰에서 전체 근무표 확인
   - 날짜 클릭 시 해당 날짜의 상세 정보 확인

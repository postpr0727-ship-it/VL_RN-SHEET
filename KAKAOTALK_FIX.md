# 카카오톡 링크 미리보기 이미지 문제 해결 가이드

## 현재 문제
카카오톡에 링크를 공유했을 때 이미지가 표시되지 않는 문제가 있습니다.

## 원인
1. **이미지 크기 부족**: 현재 `vl-logo.png`는 228x160 크기로 미리보기에 적합하지 않습니다.
2. **카카오톡 캐시**: 카카오톡은 링크 정보를 캐시하므로, 변경사항이 즉시 반영되지 않을 수 있습니다.

## 해결 방법

### 1. 카카오톡 캐시 초기화 (즉시 적용)

1. [카카오 개발자 도구 - OG 캐시 초기화](https://developers.kakao.com/tool/clear/og) 접속
2. 웹사이트 URL 입력: `https://vl-rn-sheet.vercel.app/`
3. "초기화" 버튼 클릭
4. 몇 분 후 카카오톡에서 링크를 다시 공유해보세요

### 2. 더 큰 미리보기 이미지 생성 (권장)

카카오톡 미리보기에는 최소 **800x800** 또는 **1200x630** 크기의 이미지가 필요합니다.

**이미지 준비:**
- 권장 크기: 1200 x 630 픽셀
- 파일명: `og-image.png` 또는 `og-image.jpg`
- 위치: `public/` 폴더

**이미지 생성 후:**
1. `public/og-image.png` 파일로 저장
2. `index.html`의 `og:image` 경로를 `/og-image.png`로 변경
3. 배포 후 카카오톡 캐시 초기화

### 3. 현재 이미지로 테스트

현재 `vl-logo.png`로도 작동할 수 있으므로, 먼저 캐시 초기화를 시도해보세요.

## 참고사항

- 카카오톡은 HTTPS URL의 이미지만 표시합니다
- 이미지 파일 크기는 1MB 이하 권장
- JPG, PNG 형식 모두 지원

## 단계별 체크리스트

- [ ] 카카오톡 개발자 도구에서 캐시 초기화 완료
- [ ] 배포된 사이트에서 HTML 소스 확인 (메타 태그 확인)
- [ ] 이미지 URL이 올바르게 접근 가능한지 확인
- [ ] 더 큰 이미지 생성 (선택사항)
- [ ] 변경사항 배포 후 재확인

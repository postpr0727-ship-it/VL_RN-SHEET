# 링크 미리보기 이미지 생성 가이드

## 간호사 이모지 이미지 생성 방법

### 방법 1: 온라인 도구 사용

1. **Canva** (https://www.canva.com/)
   - 1200 x 630 크기의 이미지 생성
   - 간호사 이모지 👩‍⚕️ 추가
   - 텍스트: "VL 레지던스 건강관리센터 - 간호사 근무표"
   - 저장 후 `public/og-image.png`로 저장

2. **Figma** (https://www.figma.com/)
   - 동일한 방식으로 이미지 생성

### 방법 2: HTML로 생성 후 스크린샷

1. `public/create-og-image.html` 파일을 브라우저에서 열기
2. 스크린샷 도구로 1200x630 크기로 캡처
3. `public/og-image.png`로 저장

### 방법 3: 직접 이미지 파일 제공

간호사 이모지가 포함된 이미지 파일(1200x630 권장)을 제공해주시면 바로 적용하겠습니다.

## 이미지 적용 방법

이미지 파일 준비 후:

1. `public/og-image.png` 파일로 저장
2. `index.html`의 `og:image` 경로 확인
3. 배포 후 카카오톡 캐시 초기화

## 참고사항

- 이미지 크기: 1200 x 630 픽셀 권장
- 파일 형식: PNG 또는 JPG
- 파일 크기: 1MB 이하 권장
- 배경: 투명 또는 단색 배경 가능
